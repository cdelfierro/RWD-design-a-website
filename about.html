This is the first assignment of the Coursera MOOC Responsive Web Design. It consists on a webpage that displays a list of tasks. In particular, Carlos' tasks. The tasks are in the Home page, and are represented as a list that uses Bootstrap's list group component where the completed tasks are marked with a green background and the OK icon. The tasks are clickable and links to the task details page. Currently, all the tasks in the home page, links to the same task details page.

In the task details page there is a title for the task, followed by a description, the asignee (with a little thumbnail) and a label stating if the task is finished or not.

The navigation of the site is made with two components: the Nav bar and the Breadcrumbs. The navbar shows the two main pages: The Home page and the About page and highlight the current page. The breadcrumbs are shown below the main title and shows the hierarchy of the navigation as the user goes deep in it.

For the design, the theme Slate was chosen. The main content is centered using the classes provided by Bootstrap and it is responsive according the size of the screen.
